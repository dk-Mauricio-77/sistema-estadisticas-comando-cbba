# Sistema de Gestión Estadística - Interfaz de Usuario (Frontend)

Interfaz gráfica del sistema para el Comando Departamental de la Policía.
Desarrollada con React y Vite para garantizar alto rendimiento y tiempos de carga mínimos.

## Requisitos

- Node.js v18 o superior
- PostgreSQL 14+

## Instalación y Despliegue

1. Instalar dependencias:
   ```bash
   npm install

2. Ejecutar en modo desarrollo:
   npm run dev  

3. Construir para producción:
    npm run build
